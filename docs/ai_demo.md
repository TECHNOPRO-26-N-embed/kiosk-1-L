# 自動販売機システム 操作手順書

## 1. システム概要
本システムは、飲料専用の自動販売機を模したCUI（テキストベース）アプリケーションです。利用者は飲料の購入、管理者は在庫補充が可能です。

## 2. 起動方法
1. コマンドプロンプトまたはPowerShellを開く
2. アプリケーションのあるディレクトリに移動
3. `app.exe` を実行

```
app.exe
```

## 3. メインメニュー画面
起動すると、以下のメニューが表示されます。

- 1: 購入
- 2: 在庫管理（管理者用）
- 0: 終了

番号を入力し、Enterキーを押してください。

## 4. 購入操作手順
1. メインメニューで「1」を選択
2. 商品一覧が表示されるので、購入したい商品の番号を入力
3. 購入個数を入力（1以上、在庫以下）
4. 合計金額が表示されるので、支払金額を入力（合計金額以上）
5. 購入結果（釣銭、完了メッセージ）が表示されます
6. 継続購入またはメインメニューに戻ります

## 5. 在庫管理操作手順（管理者）
1. メインメニューで「2」を選択
2. 管理者パスワードを入力
3. 商品一覧が表示されるので、補充したい商品の番号を入力
4. 補充数を入力（1以上、在庫上限50以下）
5. 補充結果が表示され、メインメニューに戻ります

## 6. 終了方法
- メインメニューで「0」を入力し、Enterキーを押すとプログラムが終了します

## 7. エラー時の対応
- 数字以外や範囲外の入力時はエラーメッセージが表示され、再入力となります
- 在庫不足や支払金額不足も同様に再入力となります
- ファイルエラー時はエラーメッセージが表示され、メインメニューに戻ります

## 8. 注意事項
- 商品数・在庫数の上限は50です
- 管理者パスワードはシステム管理者から通知されます
- 商品情報や購入履歴はCSVファイルで管理されています
