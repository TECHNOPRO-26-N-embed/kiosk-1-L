## 3. テストデータ例

- products.csv, replace_products.csv, ai_admin.conf など、正常・異常パターンのサンプルファイルを用意

## 4. 実施方法

- 各関数を単体で呼び出し、入力値ごとに期待結果を確認
- エラー時はエラーメッセージやログ出力も確認
