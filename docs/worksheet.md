## 2. 要求仕様書（顧客ヒアリング結果）

### 2.1 顧客の要望（ヒアリングメモ）

> ここに、顧客（講師）へのヒアリングで得た要望を箇条書きで記録してください。

* 販売する商品は飲料を想定する。飲料の種類・価格はチーム側で適当に決めていい。
* 飲料以外の商品は、今回は扱わない。
* 各商品の最大在庫数は、50個程度を目安とする。
* 想定外の入力や操作があっても、プログラムが止まらないようにする。
* 自動販売機の中の効果の枚数は考慮しない
* 同時に扱う商品は50種類
* 在庫は1種類あたり50個
* 季節ごとの製品入換あり

### 2.2 機能要件

| 機能ID | 機能名 | 概要 |
| :---: | :--- | :--- |
| F01 |商品一覧表示 |商品の金額と商品名の一覧を表示する |
| F02 |購入機能 |商品選択　→　個数を入力　→　支払い金額を表示　→　お金を受け取る　→　指定された商品を出す　→　お釣りの出力 |
| F03 |購入履歴保存 |csvファイルに変更を適用 |
| F04 |在庫管理 |入力をもとにcsvファイルを変更 |
